# 🛡️ 一水安全加解密助手

> **纯前端非对称加密 | 数据不上传服务器 | 自定义时效安全**

## 📋 项目简介

“一水安全加解密助手”是一款基于 Web Crypto API 开发的浏览器端数据安全工具。它利用现代浏览器的原生加密能力，在**不连接服务器**的情况下，实现对文字和文件的高强度加密。

### 💡 核心优势

*   **🔒 数据零上传**：所有加密、解密运算均在本地浏览器完成，你的数据永远不会经过第三方服务器，从根本上杜绝泄露风险。
*   **⏳ 时效性控制**：支持设置密文有效期（一次性、3小时、1天、3天、7天、永久），过期后无法解密，确保信息的时效安全。
*   **🌐 全平台通用**：无需安装软件，打开网页即可使用。支持 Windows、Mac、Linux 以及手机端（iOS/Android）。
*   **📂 格式广泛**：支持文字加密，以及图片、视频、文档等各种格式文件的加密打包（生成 `.ys` 文件）。

## 🚀 快速开始

### 1. 文字加密与解密
适用于加密聊天记录、密码、短文本等。

#### 🔐 加密流程
1. 在首页选择 **“文字加密”** 模式。
2. 输入 **4位数字PIN码**（例如：`1234`）。
3. 设置有效期（默认3天）。
4. 输入需要加密的文字内容，点击 **“加密”** 按钮。
5. 系统生成一串 Base64 密文，点击复制并发送给好友。

#### 🔓 解密流程
1. 切换到 **“文字解密”** 选项卡。
2. 输入与加密时相同的 **4位PIN码**。
3. 将收到的密文粘贴到输入框中。
4. 点击 **“解密”**，若在有效期内且密码正确，即可查看原文。

### 2. 文件加密与解密
适用于加密图片、视频或重要文档。

#### 🔐 加密流程
1. 在首页选择 **“文件加密”** 模式。
2. 点击上传区域选择文件，并输入 **PIN码**。
3. 点击 **“加密并下载 .ys”**。
4. 浏览器会自动下载一个后缀为 `.ys` 的安全文件。

#### 🔓 解密流程
1. 切换到 **“文件解密”** 选项卡。
2. 上传接收到的 `.ys` 文件。
3. 输入正确的 **PIN码**，点击 **“解密并预览/下载”**。
4. 验证通过后，可直接在网页内预览图片/视频，或点击下载原始文件。

为了让好友顺利解密，请参考以下发送建议：

| 发送场景 | 操作建议 |
| :--- | :--- |
| **微信 / QQ** | 发送文件时，务必选择加密生成的 **`.ys` 后缀文件**。**切勿**直接发送图片/视频原图，必须发送加密后的文件。 |
| **安卓手机** | 接收方下载 `.ys` 文件后，打开本工具网页，选择“文件解密”并上传该文件即可。 |
| **苹果 iOS** | 接收方在微信/QQ中点击 `.ys` 文件，选择“用其他应用打开” -> “存储到文件”，然后在 Safari 浏览器中打开本工具进行上传解密。 |

## ⚙️ 技术细节

*   **加密算法**: AES-GCM (256位)
*   **密钥派生**: PBKDF2 (SHA-256, 100,000次迭代)
*   **运行环境**: 纯前端 JavaScript (Web Crypto API)
*   **文件格式**: `.ys` (包含头部信息 + 加密数据的二进制流)

---

## ⚠️ 注意事项

1.  **PIN码遗忘无法恢复**：由于是纯本地加密，没有后台数据库存储密码。如果忘记 PIN 码或文件过期，数据将**永久无法恢复**，请务必确认好密码再发送。
2.  **时效性校验**：解密时会基于北京时间校验时效，过期文件将被系统拒绝解密。
3.  **浏览器兼容性**：推荐使用 Chrome、Edge、Firefox 或 Safari 的最新版本浏览器以获得最佳支持。
